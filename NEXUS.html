6 Autonomous Agents

Built for distance, not for smartphones

👁

BASAR (بصر) — Vision

YOLOv8 pose estimation watches a webcam and detects falls in real time. No human needed. Fires Make.com webhook within 2 seconds. Sends 15-minute stats reports to Google Sheets + Gmail.

🧠

ORACLE — AI Triage

Claude claude-sonnet-4-6 classifies every incoming message as LOW / MEDIUM / HIGH / EMERGENCY. Responds in Arabic or English. Routes to correct response pipeline automatically.

🛡️

SENTINEL — Security

Offline cybersecurity agent. Runs rule-based threat detection on all data flows. Produces a MOSCA Score (0–100) measuring patient data security. No API keys required — 100% free.

📡

BASIRA — Prediction

Scheduled Claude agent runs every 6 hours. Predicts community health demand for the next week using historical alert patterns. Logs forecasts to Supabase for clinic pre-positioning.

⚡

Make.com Pipeline

The automation backbone. Receives webhooks from all agents. Routes to Google Sheets (log), Gmail (alert email), and Twilio SMS — all in parallel, in under 10 seconds.

🔗

n8n Orchestrator

Visual workflow canvas connecting all agents. Handles WhatsApp triage via Twilio, schedules BASIRA predictions, monitors SENTINEL reports. Fully importable JSON workflow.
